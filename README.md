# Binary Tree

Java binary tree and binary search tree utilities, including traversal helpers, search/insert/delete operations, display support, and a custom `MyTreeSet` wrapper.

## Course Context

Course: Honors Data Structures  
School: The Harker School  

Honors Data Structures covered object-oriented Java programming and core data structures such as lists, stacks, recursion, trees, heaps, and algorithmic problem solving. This project focuses on recursive tree algorithms and binary search tree behavior.

## What This Demonstrates

- Recursive binary tree traversal and utility methods
- Binary search tree insertion, lookup, and deletion
- Tree-backed set abstraction
- Tree visualization and interactive tester classes
- File-backed knowledge tree data

## Project Structure

- `src/main/java/com/vinayak/TreeUtil.java` - binary tree algorithms and helpers
- `src/main/java/com/vinayak/BinaryTreeUtil.java` - search-tree operations
- `src/main/java/com/vinayak/MyTreeSet.java` - set wrapper built on a binary search tree
- `src/main/java/com/vinayak/TreeDisplay.java` - visual tree display
- `src/main/java/com/vinayak/*Tester.java` - tester and demo programs

## Run

```bash
mvn compile
java -cp target/classes com.vinayak.TreeSetTester
```
